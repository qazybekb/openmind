# Changelog

All notable changes to OpenMind will be documented in this file.

## 0.1.0 - 2026-03-26

- Publish the initial UC Berkeley release of OpenMind.
- Add 38 documented tools across Canvas, Berkeley campus data, course catalog search, profile management, PDF/web, and optional integrations.
- Harden privacy and security messaging across the CLI, docs, and website to match actual runtime behavior.
- Add local-file safety improvements: owner-only permissions for sensitive files, atomic config/profile/state writes, and explicit-user-intent gating for sensitive integrations.
- Add automated release checks for linting, tests, CLI smoke checks, tool counts, and website builds.
